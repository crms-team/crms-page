CRMS 설치 가이드

  1. 프로젝트 파일을 다운로드 받습니다.

    2. npm i

  2. Terminal에서 위 명령어를 입력해 필요한 모듈들을 설치합니다.

  3. 폴더 최상단에 .env파일을 만듭니다.

    4. REACT_APP_SERVER_URL=http://localhost:4000

  4. .env 파일안에 위와 같이 작성합니다. 

    5. npm run-script build
node server

  5. 그 후 위 명령어를 실행해 CRMS를 실행합니다.

설치 완료 후 CRMS를 사용하기 전 다음과 같은 준비 과정이 필요합니다.


AWS Key 발급 방법


  1. https://aws.amazon.com/ko/ 에 접속하고 본인 계정으로 로그인합니다.

  2. https://console.aws.amazon.com/iam/home?region=ap-northeast-2#/users 에 접근한 후 사용자 추가를 누릅니다.

  3. 사용자 이름을 작성한 후 프로그래밍 방식 액세스를 체크한 후 다음을 누릅니다.

  4. 기존 정책 직접 연결을 클릭하고 아래 표에서 해당하는 내용을 찾아 필터에서 검색합니다.

  5. 그 후 검토 페이지까지 다음을 누릅니다.

  6. 사용자 만들기를 클릭합니다.

  7. 그 후 만들어진 키에 대한 액세스 키와 비밀 액세스 키가 담겨있는 csv를 다운로드 하시면 완료입니다.

Azure Key 발급 방법


  1. 로그인을 진행한 후 https://portal.azure.com/#home에 접근합니다.

  2. 상단 검색창에 Azure Active Directory를 입력합니다.

  3. 좌측 메뉴에서 앱 등록을 클릭합니다.

  4. 클릭한 후 새 등록을 클릭합니다.

  5. 이름을 입력한 후 이 조직 디렉터리의 계정만(기본 디렉터리만 - 단일 테넌트)를 클릭하고 등록 버튼을 클릭합니다.

  6. 등록한 후 나오는 기본정보 창에서 클라이언트 ID와 테넌트 ID를 메모합니다.

  7. 그 후 좌측 메뉴에서 인증서 및 암호를 클릭합니다.

  8. 그 후 새 클라이언트 암호 버튼을 클릭하신 후 나오는 Secret key를 메모합니다.

  9. 그 후 검색창에서 구독을 입력하고 이동합니다.

  10. 본인이 만든 애플리케이션이 속한 구독을 선택합니다.

  11. 구독 ID를 복사하여 적어놓습니다.

  12. 그 후 좌측 메뉴에서 IAM을 클릭합니다.

  13. 추가버튼을 클릭한 후 역할 할당 추가 버튼을 클릭합니다. 그 후 아래 표를 보고 해당하는 역할을 고릅니다.

  14. 다음에 대한 액세스 할당을 사용자, 그룹 또는 서비스 주체로 선택한 후 등록한 앱을 선택합니다.

    15. 위 과정을 완료했다면 CRMS를 사용할 준비가 완료되었습니다! 하단에 과정을 진행해주시면 감사하겠습니다.


    CRMS 사용 방법


    1. 그 후 data 폴더에서 crms.config 파일 안에 있는 비밀번호를 확인합니다. ( 초기 설정은 1234입니다. )

    2. 사이트에 접근하고 로그인을 진행합니다.

    3. Sidebar에 Setting 페이지로 이동하여 Register Cloud Key를 클릭합니다.

    4. 원하는 벤더를 입력한 후 ID를 입력하고 중복 확인을 클릭합니다.

    5. AWS에 경우 csv 파일 안에 Access key와 Secret key에 대한 값을 입력한 후 사용자에region을 입력한 후 추가를 클릭하면 완료입니다.

    6. Azure에 경우 적어놓으신 메모에 있는 Subscription ID, Client ID, Secret Key, TenantID를 입력한 후 추가를 클릭하면 완료입니다.
